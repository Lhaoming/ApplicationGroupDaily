2.切换到D盘：$ cd d:

$ git clone (需要克隆的github网址)

进入本地项目源代码目录：

$ cd ApplicationGroupDaily

添加所有文件，如需只添加一部分文件需输入文件名:

git add . //add .
git commit -m "year-month-day"
git push

3.git原理简介

仓库：本地仓库和远程仓库(托管在网络端的仓库)

本地仓库：工作区丶版本区，其中版本区包含暂存区丶仓库区

从本地仓库将文件git到远程仓库流程：工作区----> 暂存区 ----> 仓库区 ----> 远程仓库
